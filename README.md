Customer Segmentation using K-Means Clustering

Project Overview

This project uses the K-Means clustering algorithm to segment customers based on their Annual Income and Spending Score.

Customer segmentation helps businesses understand customer behavior and target the right customers with appropriate marketing strategies.

Dataset

The dataset used in this project is the Mall Customers Dataset.
or you can download from kaggle

It contains the following columns:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Algorithm

K-Means Clustering (Unsupervised Machine Learning)

Result

Customers are divided into different clusters based on their income and spending behavior.
This helps identify groups such as high spending customers, low spending customers, and average customers.

Visualization

The clusters are visualized using a scatter plot of Annual Income vs Spending Score.
